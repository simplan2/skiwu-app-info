## Requisitos Técnicos

**Específicos de Windows:**
- Windows 10/11
- Arquitectura x64
- .NET 10.0 Desktop Runtime

**Específicos de Android:**
- Android 10.0 (API nivel 29) o superior
- Permiso de almacenamiento concedido
- 2 GB de RAM para óptimo rendimiento

Adicionalmente usted debe tener **Conexión a Internet**
